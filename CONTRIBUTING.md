# Contributing

Thanks for your interest in contributing to Knox Racquet Stringing.

## How to contribute

- **Open an issue** to suggest a change, report a bug, or ask a question.
- **Open a pull request** for small fixes (typos, docs, accessibility, etc.). For larger changes, opening an issue first is helpful so we can align before you spend time.
- **Security issues:** Please report vulnerabilities privately; see [SECURITY.md](SECURITY.md).

## Development setup

1. Clone the repo.
2. Run `npm install` and `npm run build`.
3. Open `site/index.html` in a browser or use a local server (e.g. `npx serve site`) to preview.

The project is plain HTML, CSS, and JavaScript with no framework.

## Pull requests

- Base branch: `main`.
- Keep changes focused; put unrelated edits in separate PRs.
- CI must pass (HTML validation, link check, Lighthouse, Pa11y, lockfile check, etc.).
- Match existing code style and structure.

## Code of conduct

Participation in this project is governed by our [Code of Conduct](CODE_OF_CONDUCT.md). By contributing, you agree to follow it.
